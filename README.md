# Пакет ВКР на пересдачу

Данный пакет содержит актуальную версию магистерской ВКР по теме повышения энергоэффективности систем жизнеобеспечения кинозала, а также все основные материалы, подготовленные для повторной сдачи после исправления расчётных и оформительских ошибок.

## Состав пакета

- `01_Полная_работа/ВКР_Кинозал_DCV_финал.docx` — полная исправленная версия ВКР
- `02_Части/Часть_1.docx` ... `Часть_4.docx` — работа, разбитая на части
- `03_Артефакты/Иллюстрации` — актуальные рисунки, схемы и графики, использованные в работе
- `03_Артефакты/Код/node_red_flow.json` — актуальный поток Node-RED
- `03_Артефакты/Код/ris45_node_red_code.png` — изображение кода для вставки в работу
- `03_Артефакты/Данные/results.json` — итоговые расчётные данные моделирования

## Что исправлено

- синхронизированы текст, таблицы, выводы и экономические показатели
- обновлены встроенные графики в полной версии работы
- заново подготовлена картинка кода для раздела автоматизации
- части работы приведены к структуре, удобной для сдачи преподавателю

## Основной файл

Для сдачи полной версии использовать:

`01_Полная_работа/ВКР_Кинозал_DCV_финал.docx`

Если преподаватель требует работу по частям, использовать файлы из папки:

`02_Части`
