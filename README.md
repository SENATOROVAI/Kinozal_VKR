# Kinozal_VKR

ВКР по повышению энергоэффективности систем жизнеобеспечения кинозала за счет контроля концентрации CO2 и автоматического управления вентиляцией.

## Кратко

**Задача:** снизить электропотребление вентиляционной системы кинозала без ухудшения качества воздуха.

**Как решили:** проанализированы инженерные системы кинозала, построена расчетная модель объекта в OpenStudio, сравнены шесть сценариев энергосбережения, отдельно разработана автоматизация DCV на базе датчиков CO2, ПЛК и Node-RED.

**Вывод:** управление воздухообменом по потребности снижает годовое электропотребление с 7700 до 4058 кВт·ч, экономия составляет 3642 кВт·ч/год или 47,3 %, средняя концентрация CO2 в занятые часы остается около 779 ppm, простой срок окупаемости - 5,1 года.
